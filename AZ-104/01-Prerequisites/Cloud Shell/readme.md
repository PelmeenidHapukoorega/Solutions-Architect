# Azure Cloud Shell

Browser based authenticated terminal for managing Azure resources.

## The situation: When to reach for this tool?

When you need to run commands or scripts immediately from any browser without installing software on your local device.

## The protection: What does it stop?

Blocks credential theft. When you are logged into the Portal you dont have to store sensitive passwords or Access Keys on computer drive.

## The payoff: What does it simplify?

Zero maintenance, you dont have to update CLI versions or manage Python/Node.js environments. Microsoft keeps all the tools uptadet for you, which include but not limited to Terraform, Git, Python, Go, Java etc.
